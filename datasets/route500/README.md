# route500

Dédiée aux applications cartographiques et routières à l'échelon régional, cette base **proposait** historiquement la description du réseau routier classé (500 000 km de voies). 

## Mise en garde

- Depuis l’édition 2019, ROUTE 500® correspond au réseau routier de la BD CARTO®.
- La dernière version disponible date de 2021
- Le script ci-dessous importe la version du 30/10/2019 à partir de https://files.opendatarchives.fr/professionnels.ign.fr/

## Utilisation

```bash
uv run datasets/route500/import.py
```
