# codes_postaux

Script de chargement de la [base officielle des codes postaux de La Poste](https://data.laposte.fr/datasets/laposte-hexasmal) dans une base PostGIS.

## Utilisation

```bash
uv run python -m datasets.codes_postaux.import
```

## Ressources

- [data.laposte.fr - laposte-hexasmal - Base officielle des codes postaux](https://data.laposte.fr/datasets/laposte-hexasmal)
- https://data.laposte.fr/data-fair/api/v1/datasets/laposte-hexasmal/data-files/019HexaSmal.csv
- https://data.laposte.fr/data-fair/api/v1/datasets/laposte-hexasmal/data-files/019HexaSmal-full.csv
